Voice Command Guide

"stop voice" Stops the recording immediately. (Customizable in settings)
"end voice download file" Stops recording and exports the document to Word. (Customizable in settings)
"recording top add" Moves the current text insertion point to the very top of the document.
"access style [Category]" Scrolls to a style category (e.g., "access style Typographic").
"access style [Category] [Action]" Toggles a specific format (e.g., "access style Typographic Bold"). Highlights the tool if active.
"delete text [word]" / "remove text [word]" Deletes the last word typed if it matches (e.g., "remove text Jai").
"default text" / "here empty sheet" Populates the editor with a welcome message and speaks it.
"guide" Opens this helpful voice command reference modal.
